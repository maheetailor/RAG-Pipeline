# RAG-Pipeline
Here's my 1st ever RAG Pipeline; integrated with LLM- GROQ

I have also attached 3 files which you can use to check-out the answers.
If using those file make sure you download them and save them inside the same folder and name that folder as "data".

What you have to do?
Firstly, you will have to sign up in GROQ.
Secondly, Make your own API KEY to integrate with LLM; copy that key.(Do not share it with anyone)
In the code I have written to enter the API KEY you can either enter or paste in the double quotes.
Code will run successfully.
You can do the same for ChatGPT or CLAUDE.
